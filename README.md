# Etch-a-Sketch

A browser-based drawing board built with HTML, CSS, and JavaScript.

## Features

- 16x16 default drawing grid
- Hover over squares to draw
- Change grid size from 1 to 100
- Reset board to 16x16
- Dynamic square sizing based on board width

## Technologies Used

- HTML
- CSS
- JavaScript
- Flexbox
- DOM manipulation

## What I Learned

- How to create elements dynamically with JavaScript
- How to use event listeners
- How to use Flexbox with wrapping
- How to validate user input
- How to calculate dynamic square sizes
